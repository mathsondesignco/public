### [2015-03-28](/news/2015/03/28/index.md)

# New South Wales state election, 2015. Voters in the Australian state of New South Wales go to the polls for a state election with the Liberal-National Coalition led by Mike Baird retaining government. 

Counting is underway in the NSW state election with polls pointing to a comfortable victory for the incumbent Coalition.
