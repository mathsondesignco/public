
## Privacy Policy

*the Context* aims to **not** collect any visitor information:

* the webserver does not store access or error logs
* the website does not use cookies, sessions or local storage
* the website does not use tracking or analytics
* the website does not display ads

*the Context* aims to serve an open and multi-format content:

* the website is readable in HTML, JSON, XML and Markdown
* the API is free, open to all and not rate-limited
* the website is open sourced on GitHub

*the Context* is based on infrastructure that may, or may not follow the Context principles:

* Server and DNS infrastructure is provided by [Linode, LLC](https://www.linode.com), located in New Jersey, USA. There is no reason to believe it is tracking visitors of its machines, however it's technically possible
* Domain registrar is [Domain.com, LLC](http://www.domain.com), domain service is provided by [MyDomain, Inc](https://mydomain.com)
